Blood Donation Management System Overview:-

The Blood Donation Management System is a web-based application that allows users to search for blood donors based on blood group and register themselves as donors. This project is built using PHP, MySQL, and JavaScript.

Features:-
 Search donors by blood group
 Display donor details (Name, Email, Mobile, City)
 Donor registration system
 Session-based login & logout
 Dynamic data loading using AJAX
 Update donor profile (if logged in)

Tech Stack:-
   Frontend: HTML,  CSS, JavaScript
   Backend: PHP
   Database: MySQL
    Server: Apache (XAMPP)
